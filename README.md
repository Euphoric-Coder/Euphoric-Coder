## Featured Projects

### CityCare – Smart Civic Issue Management Platform

A full-stack civic-tech ecosystem for reporting, verifying, assigning, tracking, and resolving public issues through web and mobile workflows.

**Highlights**

* Multi-role platform for Citizens, Unit Officers, Field Officers, Admins, and City Admins
* Citizen web portal, admin/staff dashboard, and React Native officer mobile app
* Geo-tagged issue reporting with images, category, priority, address, latitude, and longitude
* Duplicate detection using location proximity, category, subcategory, and text similarity
* SLA tracking, escalation handling, officer assignment, and resolution verification
* AI-assisted issue review for better classification and triaging support

**Tech Used:** Next.js, React Native Expo, ConvexDB, NextAuth, Upstash, Resend, Gemini AI, Google Maps API

**Why it stands out:**
CityCare goes beyond a basic complaint system by modeling real civic workflows such as verification, duplicate review, officer assignment, SLA monitoring, escalation, and transparent resolution tracking.

### MediCura HMIS – Hospital Management Information System

A full-stack hospital management platform that connects patients, doctors, receptionists, pharmacists, and admins in one centralized workflow.

**Highlights**

* Role-based dashboards for Patients, Doctors, Pharmacists, Receptionists, and Admins
* Appointment booking and scheduling workflow
* Doctor consultation and prescription management
* Prescription-to-pharmacy flow with inventory support
* Admin dashboard for users, appointments, consultations, prescriptions, and pharmacy activity
* Secure role-based authentication using NextAuth
* IST-aware scheduling for reliable Indian hospital workflows

**Tech Used:** Next.js, JavaScript, Tailwind CSS, ShadCN UI, Drizzle ORM, PostgreSQL NeonDB, NextAuth, jsPDF

**Why it stands out:**
MediCura is designed around real hospital operations instead of disconnected modules, connecting appointments, consultations, prescriptions, pharmacy tasks, and admin monitoring into one structured system.
